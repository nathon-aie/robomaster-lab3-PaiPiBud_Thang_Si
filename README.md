## โครงสร้างโปรเจกต์
- `main.py`: สคริปต์รันหุ่นยนต์หลัก (Entry Point) ดึงโค้ดทุกส่วนมารวมกันที่นี่
- `config/`: โฟลเดอร์เก็บไฟล์ตั้งค่าคงที่
  - `settings.yaml`: เก็บพารามิเตอร์หุ่น
- `src/`: โฟลเดอร์ซอร์สโค้ดหลักสำหรับควบคุมฮาร์ดแวร์ (Modules)
  - `config_loader.py`: อ่านและแปลงไฟล์ `settings.yaml`
  - `chassis.py`: ควบคุมล้อ Mecanum
  - `gimbal.py`: ควบคุมป้อมปืน/มุมกล้อง
  - `vision.py`: สตรีมภาพและประมวลผลกล้อง
  - `logger.py`: เขียนค่าจากเซนเซอร์ลงไฟล์ CSV แบบเรียลไทม์
- `data/`: เก็บไฟล์ข้อมูล
  - `raw/`: ข้อมูลดิบจากหุ่นยนต์
  - `processed/`: ข้อมูลที่ผ่านการประมวลผล/กรองแล้ว
- `analysis/`: วิเคราะห์ผลการวิ่ง
  - `analyze_logs.ipynb`: โน้ตบุ๊กสำหรับวิเคราะห์และพลอตกราฟ
  - `plots/`: รูปภาพกราฟที่ได้จากการวิเคราะห์
